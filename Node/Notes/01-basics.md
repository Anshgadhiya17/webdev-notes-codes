# 📘 Node.js Basics

## What is Node.js?

Node.js is a runtime environment that allows JavaScript to run on the server using the V8 engine.

---

## Features

- Non-blocking (asynchronous)
- Event-driven architecture
- Fast execution
- Single-threaded

---

## Example

```js
console.log("Hello Node");
```

---

## Modules

```js
const fs = require("fs");
```

---

## Event Loop

Node uses an event loop to handle async operations efficiently.

---

## Use Cases

- APIs
- Real-time apps
- Microservices
